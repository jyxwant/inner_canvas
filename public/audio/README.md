# 背景音乐文件

## 推荐下载的音乐类型

从 https://pixabay.com/zh/music/search/%e8%bd%bb%e9%9f%b3%e4%b9%90/ 下载以下类型的音乐：

### 1. Neutral（中性/平静）
- **关键词搜索：** "ambient", "calm", "peaceful", "meditation"
- **推荐风格：** 轻柔的钢琴、环境音、自然声音
- **文件名：** `neutral.mp3` 或 `neutral.ogg`

### 2. Mystery（神秘）
- **关键词搜索：** "mysterious", "dark ambient", "atmospheric"
- **推荐风格：** 低沉的氛围音乐、轻微的不和谐音
- **文件名：** `mystery.mp3` 或 `mystery.ogg`

### 3. Tension（紧张）
- **关键词搜索：** "tension", "suspense", "dramatic"
- **推荐风格：** 紧张的氛围音乐、轻微的不安感
- **文件名：** `tension.mp3` 或 `tension.ogg`

### 4. Melancholy（忧郁）
- **关键词搜索：** "sad", "melancholic", "emotional", "piano sad"
- **推荐风格：** 悲伤的钢琴曲、慢节奏的弦乐
- **文件名：** `melancholy.mp3` 或 `melancholy.ogg`

### 5. Epiphany（顿悟）
- **关键词搜索：** "uplifting", "inspirational", "hopeful", "bright"
- **推荐风格：** 明亮、充满希望的轻音乐
- **文件名：** `epiphany.mp3` 或 `epiphany.ogg`

## 下载步骤

1. 访问 https://pixabay.com/zh/music/search/%e8%bd%bb%e9%9f%b3%e4%b9%90/
2. 根据上面的关键词搜索对应的音乐
3. 选择适合的音乐（建议时长 2-5 分钟，可以循环播放）
4. 点击下载按钮
5. 将下载的文件重命名为对应的文件名（如 `neutral.mp3`）
6. 将文件放到这个目录：`public/audio/`

## 文件格式

- 支持格式：MP3, OGG, WAV
- 推荐格式：MP3（文件小）或 OGG（音质好）
- 建议比特率：128-192 kbps（平衡文件大小和音质）

## 注意事项

- 确保音乐没有版权问题（Pixabay 的音乐都是免费可商用的）
- 音乐应该是循环播放友好的（首尾可以无缝衔接）
- 音量应该相对一致，避免某些音乐太大声或太小声

## 快速开始

1. **下载音乐文件**（至少下载 `neutral.mp3` 作为默认音乐）
2. **重命名文件**为对应的 mood 名称（如 `neutral.mp3`）
3. **放到目录**：将文件放到 `public/audio/` 目录
4. **重启应用**：重新运行 `npm run dev`

如果某个 mood 的音乐文件不存在，应用会显示警告但不会崩溃，只是不会播放该 mood 的音乐。
